Go关键字

Nov 2, 2016 | Golang

Go关键字

  • var const 声明变量与常量
  • if else switch case break continue for goto default fallthrough 流程条件控制语句
  • package import 声明包名和导入
  • func 声明函数方法
  • interface 声明接口
  • struct 声明结构
  • type 声明自定义类型
  • map 声明map类型数据
  • range 读取slice/map/channel数据
  • return 返回
  • select 选择不同的channel
  • chan 声明channel
  • defer 在函数最后执行
  • go 并发执行
    共计25枚关键字，全是小写字母

Go注释

  • // 单行注释
  • /* */ 多行注释

部分基础知识点

  • go程序入口为package mian中的func main函数，一个可执行程序，有且只有一个package mian
  • 通过import导入其他的非main包，所有引用的包，都必须被使用，否则编译会提示错误，避免了引用多余无用的包
  • 在函数体外var声明的为全局变量
  • 通过type关键字来进行struct和interface的声明
  • import，var与const可以同时导入和声明多个
  • import可以针对不同的包名设置不同的别名
  • go语言使用大小写来区分 常量、变量、类型、接口、结构与函数的调用范围，首字母小写为private，首字母大写为public
